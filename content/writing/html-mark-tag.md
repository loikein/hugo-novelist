The `<mark>` tag defines text that should be marked or highlighted for reference or notation purposes.

```
<p>
  <mark>Highlighted text</mark>
</p>
```
The default background color of `<mark>` is yellow. You can style `<mark>` with CSS:

```
mark {
  background: blue;
  color: white;
}

```
